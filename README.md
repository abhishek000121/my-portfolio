# 🚀 Personal Portfolio & Digital Assets

An interactive, AI-optimized portfolio showcasing my frontend skills and digital widgets.

### 🛠 Tech Stack
* **Frontend:** HTML5, CSS3, JavaScript (ES6+)
* **Framework:** Bootstrap (for responsiveness)
* **AI Tools:** Optimized using **DeepSeek** and **Blackbox AI** for clean code architecture.

### ✨ Key Features
* **Interactive Profile Card:** Dynamic hover effects and social links.
* **Live Digital Clock:** Real-time synchronization using JS `setInterval`.
* **Smart Alert System:** Custom-built notification system for user interaction.
* **SEO Optimized:** Structured semantic HTML for better search rankings.
